# Wireframe Structure

## 1. Homepage Wireframe

### Header
- Logo or site name on the left
- Navigation on the right:
  - Startseite
  - Coaching-Übersicht
  - Profil
  - Kontakt
- Primary CTA button: Termin anfragen

### Hero Section
- Short headline: psychologisch fundiertes Life- und Mental-Coaching
- Supporting text: mehr Klarheit, innere Stärke und Balance im Alltag
- Primary CTA: Termin anfragen
- Secondary CTA: Angebote ansehen
- Optional visual: calm portrait or abstract coaching-related image

### Trust Strip
- Short credibility points in a horizontal row
- Beispiele:
  - Psychologisch fundiert
  - Online via Zoom
  - Langjährige Erfahrung
  - Klare Preise

### Key Services Section
- Grid of 4 or 5 service cards
- Each card includes:
  - Service title
  - 1-sentence summary
  - Duration or format
  - Link to detail page
- Cards:
  - Life Coaching
  - Raucherentwöhnung
  - Alkohol selbstbestimmt
  - Stressbewältigung
  - Autogenes Training

### About Preview Section
- Short introduction to the coach
- Photo or portrait on one side
- Key credentials on the other side
- Link to full Profil page

### Process Section
- Simple 3-step layout
  1. Anfrage senden
  2. Erstgespräch vereinbaren
  3. Coaching starten

### Value Section
- Why this coaching approach works
- Focus points:
  - Empathisch
  - Lösungsorientiert
  - Alltagstauglich
  - Psychologisch fundiert

### Closing CTA Section
- Strong invitation to get in touch
- Primary CTA: Termin anfragen
- Short reassurance line about online coaching

### Footer
- Impressum
- Datenschutz
- AGB / Honorar
- Contact details

## 2. Service Page Wireframe
Example page: Raucherentwöhnung

### Header
- Same as homepage for consistency

### Intro Hero
- Page title: Raucherentwöhnung
- Short promise: Rauchfrei mit Hypnose - jetzt auch online möglich
- Primary CTA: Termin anfragen
- Optional supporting visual

### Problem / Need Section
- Explain the user situation
- Why quitting is difficult
- Emotional and practical context

### Offer Summary Section
- Clear description of the program
- What the user gets
- Format: online sessions
- Total duration
- Price

### Process Section
- Step-by-step explanation of the program
  - Analyse des Rauchverhaltens
  - Erste Hypnose
  - Ressourcen stärken
  - Rauchstopp vorbereiten
  - Stabilisierung

### Method Section
- List of methods used
  - Verhaltenstherapie
  - Mental-Coaching
  - Hypnose nach Milton Erickson

### Benefits Section
- What changes the user can expect
- Example benefits:
  - mehr Kontrolle
  - mehr Motivation
  - nachhaltige Veränderung
  - individuelle Begleitung

### Offer Details Section
- Sitzungsanzahl
- Dauer je Sitzung
- Gesamtumfang
- Preis
- Optional material included

### FAQ / Reassurance Section
- Common concerns and answers
- Example topics:
  - Ist Online-Coaching wirksam?
  - Wie schnell kann ich starten?
  - Was brauche ich für die Sitzung?

### Final CTA Section
- Short closing statement
- Primary CTA: Termin anfragen
- Secondary CTA: Zur Übersicht

### Footer
- Same as homepage

## 3. Layout Notes
- Use a single-column flow on mobile
- Use 2-column layouts only for desktop sections with image and text
- Keep each section visually distinct with generous spacing
- Limit long paragraphs to short readable blocks
- Repeat the CTA after major content sections
- Keep the tone calm, clear, and reassuring
